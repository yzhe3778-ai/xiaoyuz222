# /image - 配图助手快捷入口

## 触发后自动执行

1. **读取配图助手技能文件**
   - 查看 `C:\Users\ruoyu\.gemini\antigravity\skills\image-assistant\SKILL.md` 获取完整指令

2. **询问用户四项必要信息**（如用户未提供）：
   - 要配图的内容（一段、一个小节、或整篇文章）
   - 用在哪里 + 观看距离（PPT投影远看 / 手机近看 / 海报）
   - 谁来看（小白/从业者/老板/学生…）
   - 偏好：更"少字清爽"还是更"信息密度"

3. **按阶段执行**：
   - 阶段1：需求澄清 → 确认内容/场景/受众/字多字少
   - 阶段2：配图规划 → 拆内容，定图清单（几张/每张讲什么）
   - 阶段3：文案定稿 → 逐字定"图上写什么"
   - 阶段4：提示词封装 → 输出可直接复制的生图提示词
   - 阶段5：迭代润色 → 根据反馈减字、换隐喻

## 快速使用示例

```
/image
@[我的文章.md] 帮我做几张配图，用于公众号推文，读者是普通用户，偏好少字清爽
```

```
/image 这段内容帮我做个封面图
```

## 默认设置

- **尺寸**: 16:9 横版（除非用户要求竖版）
- **风格**: 奶油纸底 + 彩铅水彩手绘 + 轻涂鸦，趣味但干净
- **原则**: 每张图一个核心信息，大字号、少字短句、避免密集小字
