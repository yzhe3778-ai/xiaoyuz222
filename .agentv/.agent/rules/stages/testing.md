# 测试阶段规则

**阶段标识**：TESTING
**目标**：全面测试，确保质量
**预计时长**：2-3 周

---

## 规则优先级

### 🟢 所有规则都是高优先级

#### 测试覆盖

- ✅ 整体覆盖率 ≥ 80%
- ✅ 关键逻辑 ≥ 95%
- ✅ API 必须有集成测试
- ✅ 核心流程必须有 E2E 测试

#### 性能测试

- ✅ 页面加载 < 3s
- ✅ API 响应 < 200ms (p95)

#### 安全测试

- ✅ 输入验证完整
- ✅ 安全扫描通过
- ✅ 依赖无高危漏洞

---

## 代码审查

- ✅ 至少 1 人审查
- ✅ CI 检查必须通过
- ✅ 所有问题必须解决

---

## 提示词模板

```
当前处于测试阶段。

请对 [模块] 进行全面测试：
1. 测试覆盖率检查
2. 性能测试

4. 边界情况测试
```
