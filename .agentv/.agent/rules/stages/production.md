# 生产阶段规则

**阶段标识**：PRODUCTION
**目标**：稳定运行，快速响应
**持续时间**：长期

---

## 规则优先级

### 🔴 所有规则都是最高优先级

#### 代码变更

- ✅ 所有变更必须有完整测试
- ✅ 必须有详细变更日志
- ✅ 必须有回滚方案
- ✅ 必须经过 Code Review

#### 监控日志

- ✅ 监控指标完整
- ✅ 日志记录规范
- ✅ 告警规则合理

#### 发布流程

- ✅ 测试环境先验证
- ✅ 灰度发布
- ✅ 发布审批

---

## 审批要求

| 变更类型 | 审批要求 |
| -------- | -------- |
| Bug 修复 | 1 人审核 |
| 小功能   | 2 人审核 |
| 大功能   | 团队讨论 |
| 架构变更 | 全员评审 |
